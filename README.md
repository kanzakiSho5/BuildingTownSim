#卒業制作 - 街づくりゲームの表現

## 概要
Unityを使ったグリッドベースではない街づくりゲームの表現方法の研究を行った。

## 環境
Unity 2019.4.2f1

## 実装内容
- 自由度の高い道をどうやったらUnityで表現できるか
- 交差点の表現
- ゲーム内で直観的に道を敷くにはどうしたらいいか
- 道に沿った建物の建築

## 追加アセット
プロジェクトには追加されていないため、追加が必要。
- Asphalt_materials
- 8K Skybox Pack Free
- LowpolyStreetPack
- Real Stars Skybox
- Japanese_Street
